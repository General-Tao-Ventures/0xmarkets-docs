# Cartha

**Cartha Subnet (SN35)** is the decentralized liquidity backbone for 0xMarkets, built on Bittensor. 0xMarkets is a **multi-asset decentralized perpetual futures exchange** that offers traders up to **500x leverage** on currencies, commodities, crypto, and other RWAs in a permissionless manner.

**Repositories:**
- **Miner CLI**: [cartha-cli](https://github.com/General-Tao-Ventures/cartha-cli)
- **Validator**: [cartha-validator](https://github.com/General-Tao-Ventures/cartha-validator)

## What is Cartha?

Cartha is a Bittensor subnet where miners act as **Liquidity Providers (LPs)** for the 0xMarkets DEX. It provides:

- **USDC-only collateral** with one vault per market for risk isolation
- **ALPHA emissions** based on deposit score (time × amount locked)
- **60% of trading fees** distributed to miners
- **Liquidation monitoring** by validators with slashing penalties

## What Can You Do on Cartha?

### For Miners (Liquidity Providers)

- **Deposit USDC** into designated liquidity vaults to earn rewards
- **Choose specific vaults/pairs** to support (BTC/USD, ETH/USD, GOLD/USD, EUR/USD, GBP/USD, JPY/USD)
- **Earn ALPHA emissions** based on your deposit score (time × amount)
- **Earn 60% of trading fees** generated by the 0xMarkets protocol
- **Use the Cartha Interface** for a streamlined lock/unlock experience
- **View and manage positions** through the web interface
- **Extend locks and top up** existing positions
- **Commit funds by epoch durations** for optimized returns

### For Validators

- **Monitor markets** for liquidation events
- **Execute liquidations** when margin requirements are breached
- **Earn 20% of liquidation fees** for successful executions
- **Post ALPHA collateral** to participate in liquidation execution
- **Maintain validator infrastructure** with slashing penalties for missed liquidations

### For Developers

- **Integrate with smart contracts** on Base Mainnet
- **Use the Cartha Interface** for testing frontend integration
- **Develop integration tools** and automation scripts
- **Use API endpoints** and verifier functionality
- **Build monitoring dashboards** and analytics tools

## Network Configuration

### Network Details

- **Bittensor Network**: `finney` (Bittensor mainnet)
- **Subnet UID**: `35` (Cartha subnet)
- **EVM Chain**: Base Mainnet (Chain ID: `8453`)
- **Lock UI URL**: `https://cartha.finance` (for managing locks and positions)

### Token Requirements

- **TAO**: Required for subnet registration
- **ETH**: Required for gas fees on Base Mainnet
- **USDC**: Required for liquidity provision
